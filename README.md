# Even & Odd Number Generator

A simple JavaScript web app that generates random 6-digit even or odd numbers based on user selection.

## Features
- Generate random 6-digit EVEN numbers
- Generate random 6-digit ODD numbers
- Instant result without page reload
- Clean and beginner-friendly logic

## Technologies Used
- HTML
- CSS
- JavaScript (Vanilla)

## How It Works
- Uses Math.random() to generate numbers
- Uses modulo (%) to determine even or odd
- Updates the DOM dynamically
